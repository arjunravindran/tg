# Tg 0.8.0\n\nMaintenance and backport release focused on usability, reliability, and packaging readiness.\n\n## Highlights\n\n- Added paperstrip zoom controls with improved centering and shift behavior.\n- Added tic/toc-aware event rendering and amplitude history overlay.\n- Added top-bar audio input device selector and sample-rate selector.\n- Added automatic runtime fallback to 44.1 kHz when selected rates are unsupported.\n- Improved low-BPH handling using a BPH-aware period guard.\n- Extended snapshot persistence for trace zoom, tic/toc metadata, and amplitude history.\n- Hardened Windows MSI packaging script with prerequisite checks and auto-WXS generation.\n\n## Notes\n\n- Config now persists `audio_device` and `audio_rate`.\n- Existing snapshots remain supported via serializer compatibility paths.\n- Windows MSI can be generated with `./packaging/make_msi.sh /path/to/resources`.\n
